# website
main website for sjifire.org

## tech
- static site generator: [Eleventy](https://www.11ty.dev/)
- cms: [NetlifyCMS](https://www.netlifycms.org/)
- host: [Netlify](https://www.netlify.com/)

## getting started
1. clone repo
2. run `npm install`
3. run `npm start`
